Sīc lucūtus - universal language learning mobile application

Sīc locūtus is a language-learning Android application inspired by exercises from various textbooks. It was created to help drill vocabulary and grammar during commutes.

The app is language-agnostic, meaning that language content is separate from the app itself. Teachers can create content for their subject according to specification and students may download and parse content for usage locally. Features for various languages may be added over time.

Project can be found at GitHub github.com/arseniuss/linguae
